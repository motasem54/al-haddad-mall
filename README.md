# الحداد مول | Al-Haddad Mall

موقع سوبرماركت فخم باللونين الأحمر والأسود

## المميزات

- تصميم عصري وفخم باللونين الأحمر والأسود مع لمسات ذهبية
- واجهة سهلة الاستخدام بالعربية (RTL)
- أقسام متعددة للمنتجات
- تصميم متجاوب لجميع الشاشات
- بني باستخدام Next.js 15 و Tailwind CSS

## التثبيت

```bash
# Clone the repository
git clone https://github.com/motasem54/al-haddad-mall.git
cd al-haddad-mall

# Install dependencies
pnpm install
# or
npm install

# Run development server
pnpm dev
# or
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح لرؤية النتيجة.

## التقنيات المستخدمة

- **Next.js 15** - React Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **Turbopack** - Fast Development
